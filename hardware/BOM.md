# Bill of Materials (BOM)

This document lists the hardware components required to build the ESP32 RFID IoT access control system.
Este documento lista los componentes necesarios para construir el sistema IoT RFID basado en ESP32.

---

## Components / Componentes

| Item | Component / Componente | Description / Descripción                                      | Quantity |
| ---- | ---------------------- | -------------------------------------------------------------- | -------- |
| 1    | ESP32 DevKit           | ESP32 development board / placa de desarrollo ESP32            | 1        |
| 2    | RFID RC522             | MFRC522 RFID reader module / módulo lector RFID                | 1        |
| 3    | RFID Cards / Tags      | Compatible RFID cards or key fobs / tarjetas o llaveros RFID   | 1+       |
| 4    | LCD 16x2 I2C           | LCD display with I2C interface / pantalla LCD con interfaz I2C | 1        |
| 5    | LEDs                   | Green and red LEDs / LEDs rojo y verde                         | 2        |
| 6    | Buzzer                 | Active buzzer module / módulo buzzer activo                    | 1        |
| 7    | Push Button            | Momentary push button / botón pulsador                         | 1        |
| 8    | Resistors              | 220Ω resistors for LEDs / resistencias 220Ω para LEDs          | 2        |
| 9    | Jumper Wires           | Male-to-male / male-to-female wires / cables jumper            | Several  |
| 10   | Breadboard             | Standard breadboard / protoboard                               | 1        |

---

## System Overview / Descripción del sistema

The hardware setup includes an ESP32 microcontroller connected to an RFID reader, LCD display, LEDs, and a buzzer.
El sistema utiliza un microcontrolador ESP32 conectado a un lector RFID, una pantalla LCD, LEDs y un buzzer.

The device connects to WiFi and sends events to a serverless backend implemented using Google Apps Script.
El dispositivo se conecta a WiFi y envía eventos a un backend serverless implementado con Google Apps Script.

---

## Future Hardware Improvements / Mejoras futuras

Future versions may include:
Versiones futuras pueden incluir:

* custom PCB design / diseño de PCB personalizado
* device enclosure / carcasa del dispositivo
* additional sensors / sensores adicionales
